## 使用簡介
1. 無須註冊或登入，也可以查詢目前資料庫內由其他使用者建立的書籍資料
2. 可以註冊及登入會員
3. 登入後會顯示目前已有的藏書量
4. 登入後，可直接使用查詢後的書籍資料，新增至自己的藏書
5. 直接從零開始新增書籍資料，並新增至自己的藏書
6. 管理員後台，可以修改會員等級、進行停權
